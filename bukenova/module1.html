Задание MPIBegin3

Условие: В процессах четного ранга (включая главный) ввести целое число, в процессах нечетного ранга ввести вещественное число. В каждом процессе вывести удвоенное значение введенного числа.

#include 
#pragma hdrstop
#include "pt4.h"
#include "mpi.h"

void Solve()
{
    Task("MPIBegin3");
    int flag;
    MPI_Initialized(&flag);
    if (flag == 0)
        return;
    int rank, size;
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    
    if (rank % 2 == 0) {
        int n;
        pt >> n;
        pt << 2 * n;
    } else {
        double a;
        pt >> a;
        pt << 2 * a;
    }
}

Пошаговое выполнение:

  • Инициализация MPI и получение ранга процесса
  • Проверка четности ранга для определения типа данных
  • Процессы с четным рангом работают с целыми числами
  • Процессы с нечетным рангом работают с вещественными числами
  • Каждый процесс независимо выполняет ввод и вывод

Задание MPIBegin5

Условие: В каждом процессе дано целое число N(>0) и набор из N чисел. В процессах четного ранга (включая главный) вывести сумму чисел из данного набора, в процессах нечетного ранга вывести среднее арифметическое чисел из данного набора.

#include 
#pragma hdrstop
#include "pt4.h"
#include "mpi.h"

void Solve()
{
    Task("MPIBegin5");
    int flag;
    MPI_Initialized(&flag);
    if (flag == 0)
        return;
    int rank, size;
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    
    int n;
    pt >> n;
    double sum = 0;
    for (int i = 0; i < n; i++) {
        double x;
        pt >> x;
        sum += x;
    }
    
    if (rank % 2 == 0) {
        pt << sum;
    } else {
        pt << sum / n;
    }
}

Пошаговое выполнение:

  • Инициализация MPI и получение информации о процессах
  • Чтение количества чисел N
  • Чтение N чисел и вычисление их суммы
  • Процессы с четным рангом выводят сумму
  • Процессы с нечетным рангом выводят среднее арифметическое
  • Не требуется обмен данными между процессами

Задание MPIBegin6

Условие: В каждом процессе дано целое число N(>0) и набор из N чисел. В подчиненных процессах четного ранга вывести сумму чисел из данного набора, в процессах нечетного ранга вывести среднее арифметическое чисел из данного набора, в главном процессе вывести произведение чисел из данного набора.

#include 
#pragma hdrstop
#include "pt4.h"
#include "mpi.h"

void Solve()
{
    Task("MPIBegin6");
    int flag;
    MPI_Initialized(&flag);
    if (flag == 0)
        return;
    int rank, size;
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    
    int n;
    pt >> n;
    double product = 1;
    double sum = 0;
    for (int i = 0; i < n; i++) {
        double x;
        pt >> x;
        product *= x;
        sum += x;
    }
    
    if (rank == 0) {
        pt << product;
    } else if (rank % 2 == 0) {
        pt << sum;
    } else {
        pt << sum / n;
    }
}

Пошаговое выполнение:

  • Инициализация MPI и получение ранга процесса
  • Чтение количества чисел и самого набора
  • Вычисление суммы и произведения всех чисел
  • Главный процесс (ранг 0) выводит произведение
  • Подчиненные процессы с четным рангом выводят сумму
  • Подчиненные процессы с нечетным рангом выводят среднее арифметическое
  • Каждый процесс работает независимо
