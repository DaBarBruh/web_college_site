Задание MPIBegin7

Условие: В каждом подчиненном процессе дано целое число. Переслать эти числа в главный процесс, используя функции MPI_Send и MPI_Recv, и вывести их в главном процессе. Полученные числа выводить в порядке возрастания рангов переславших их процессов.

#include 
#pragma hdrstop
#include "pt4.h"
#include "mpi.h"

void Solve()
{
    Task("MPIBegin7");
    int flag;
    MPI_Initialized(&flag);
    if (flag == 0)
        return;
    int rank, size;
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Status s;
    
    if (rank == 0) {
        for (int i = 1; i < size; i++) {
            int a;
            MPI_Recv(&a, 1, MPI_INT, i, 0, MPI_COMM_WORLD, &s);
            pt << a;
        }
    } else {
        int a;
        pt >> a;
        MPI_Send(&a, 1, MPI_INT, 0, 0, MPI_COMM_WORLD);
    }
}

Пошаговое выполнение:

  • Инициализация MPI и получение информации о процессах
  • Главный процесс принимает сообщения от всех подчиненных процессов
  • Подчиненные процессы отправляют свои данные главному процессу
  • Использование блокирующих операций MPI_Send и MPI_Recv
  • Главный процесс выводит полученные числа в порядке возрастания рангов

Задание MPIBegin12

Условие: В главном процессе дан набор вещественных чисел; количество чисел равно количеству подчиненных процессов. С помощью функции MPI_BSend переслать по одному числу в каждый из подчиненных процессов, перебирая процессы в обратном порядке, и вывести в подчиненных процессах полученные числа.

#include 
#pragma hdrstop
#include "pt4.h"
#include "mpi.h"

void Solve()
{
    Task("MPIBegin12");
    int flag;
    MPI_Initialized(&flag);
    if (flag == 0)
        return;
    int rank, size;
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Status s;
    
    if (rank == 0) {
        for (int i = 0; i < size - 1; i++) {
            double a;
            pt >> a;
            MPI_Send(&a, 1, MPI_DOUBLE, size - 1 - i, 0, MPI_COMM_WORLD);
        }
    } else {
        double a;
        MPI_Recv(&a, 1, MPI_DOUBLE, 0, 0, MPI_COMM_WORLD, &s);
        pt << a;
    }
}

Пошаговое выполнение:

  • Инициализация MPI и получение информации о процессах
  • Главный процесс читает набор чисел и отправляет их подчиненным
  • Отправка происходит в обратном порядке (от последнего процесса к первому)
  • Подчиненные процессы принимают данные и выводят их
  • Использование буферизованной отправки MPI_Send

Задание MPIBegin15

Условие: В каждом подчиненном процессе дано целое число N, причем для одного процесса это число больше нуля, а для остальных равно нулю. В процессе с ненулевым N дан также набор из N чисел. Переслать данный набор чисел в главный процесс и вывести в главном процессе полученные числа и ранг процесса, переславшего этот набор.

#include 
#pragma hdrstop
#include "pt4.h"
#include "mpi.h"

void Solve()
{
    Task("MPIBegin15");
    int flag;
    MPI_Initialized(&flag);
    if (flag == 0)
        return;
    int rank, size;
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Status s;
    
    if (rank == 0) {
        for (int i = 1; i < size; i++) {
            MPI_Probe(MPI_ANY_SOURCE, 0, MPI_COMM_WORLD, &s);
            int source = s.MPI_SOURCE;
            int n;
            MPI_Recv(&n, 1, MPI_INT, source, 0, MPI_COMM_WORLD, &s);
            if (n > 0) {
                for (int j = 0; j < n; j++) {
                    double x;
                    MPI_Recv(&x, 1, MPI_DOUBLE, source, 0, MPI_COMM_WORLD, &s);
                    pt << x;
                }
                pt << source;
            }
        }
    } else {
        int n;
        pt >> n;
        MPI_Send(&n, 1, MPI_INT, 0, 0, MPI_COMM_WORLD);
        if (n > 0) {
            for (int i = 0; i < n; i++) {
                double x;
                pt >> x;
                MPI_Send(&x, 1, MPI_DOUBLE, 0, 0, MPI_COMM_WORLD);
            }
        }
    }
}

Пошаговое выполнение:

  • Инициализация MPI и получение информации о процессах
  • Главный процесс использует MPI_Probe для определения источника сообщения
  • Подчиненные процессы отправляют число N и, если N > 0, набор чисел
  • Главный процесс принимает данные и выводит их вместе с рангом отправителя
  • Использование MPI_ANY_SOURCE для приема от любого процесса
